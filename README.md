-    `MAVEN 3.3.9 + `
   
-    `Postgres | mySql `

## installation

```xml
  ./compile.sh
``` 
A jar **jaxy-thorntail.jar** is created and copied into the folder **[jaxy](https://github.com/rac021/Jaxy/tree/master/jaxy)**
